# 关于丹麦烟酒
## 烟
- 常规香烟一般超市收银柜台都有售，目前售价58-68dkk，折合人民币60-70元左右(2024-10)。
- 烟叶和烟纸也是蛮多人的选择。
- 室内大多不能抽烟（包括一些酒吧）。
- 街道上/建筑门口和国内无异，地上也常见烟头。
## 酒
下文中酒未说明都指代烈酒。
### 法律
- 16岁及以上：可以购买酒精浓度在16.5%以下的饮品（如啤酒和葡萄酒）
- 18岁及以上：可以购买任何酒精饮料，包括烈酒
### 购买
- 常见烈酒价格约等于或稍贵于国内。
- 购于超市，如Netto, Fotex etc. 时常有促销。
- 购于实体酒行，非常丰富的选择。有的主营葡萄酒，有的烈酒。我镇上的LYNGBY VINKÆLDER也有丰富的苦精/利口酒/苦酒 etc.
- 购于网站：线上酒行、酒厂官网。
### 其他
- 酒行售的汤力水品质不错，单价20kr+ 。
- 便宜的玻璃杯具比较少。像鸡尾酒玻璃杯具我一般在temu / aliexpress购买或者放托运箱子带来（temu上我居然找不到柯林杯）
- Shaker 有朋友在这边购入过一款家用非专业的，密封性差，用料不扎实。
- 冰，因为水垢多，自己冻，最好滤水器过一遍再烧一遍。
- 普遍青柠3.5kr/颗，柠檬4kr/颗。青柠品质特不错。
- 薄荷，罗勒之类超市都可以买到，有新鲜盆栽形式 15-20 kr左右。
- 丹麦找不到demerara
