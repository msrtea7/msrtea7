# 丹麦 Radius Distillery

## 基本信息
◎一个手工金作坊，2016年正式成立，在Zealand岛南部，距离哥哈市区1h车程左右。

◎在一个私人社区里，被农场和种植业场子围绕。主要生产地改造自一个历史悠久的马厩。

◎在1层蒸馏区，目前有一个封装流水线，一个陈酿区和大小两个塔式蒸馏器，体积大概是120L, 50L(记不太清了)
◎参数：
- base liquor 用96度伏特加
- 杜松子来自东欧国家，例如波兰
- 风味物大多选自本地，部分进口
- 陈酿在阁楼，桶子蛮丰富，像是Oloroso 雪梨桶， Sherry PX桶，Jack Daniel的波本桶, ects.
	
## 产品线：
- 金酒(主要)
- 阿夸维特(akvavit)
- 白兰地
- 单桶威士忌
- 利口酒，葡萄酒，汤力水等
	
Gin Batch 018 获2018旧金山世界烈酒大赛银奖。
非常实验的酒厂，目前出品仅金酒已经到了47种，基本不返场，在售12款左右。
	
## 特征
◎苹果。非常多用在酒厂的金酒和白兰地中。来自邻居种植园的苹果，来自农场主的爷爷的爷爷整的old Danish apple varieties。自己家的后院还有3000棵年轻的预备军。

◎极简。例如金酒，基本上风味物选择会控制在个位数，个人体验是非常明确的风味表达。可能是得益于此，在酒标上做了非常棒的风味物可视化。
	
## 风味简单分享
- Batch 045 [薄荷，柚子]
    明亮，优雅的柚子风味，在嗅觉上很亮眼。喝的时候像有感受到薄荷的香不是气味的那种，而是咀嚼。带有一点点点点神奇的土壤的感受
	
- Batch 044 [黄瓜,沙棘果,啤酒花]
    太清爽了，summer time。像是有青色的寒气对着鼻子，非常明确的黄瓜，持续整个过程，一丝丝甘，沙棘果带来的一些“酸”感在结尾非常舒适
	
- Batch 047 [... 当归根 ...]
    这个我是真喜欢，做的海军金。大地，土壤，棕黄。像是和将军对话。味道一下让我想起云南漫游者出的探险力量，也是做的海军金，并且两者的瓶身是同个色系，太巧了。
	
还有一个鸢尾风味的忘了哪个批次的了，很香。轻盈，淡紫色的风味，轻盈，像在喉咙做spa。蛮多款金酒试下来，简单总结就是“干净” + “所见即所得”，极简的清晰的风味呈现。
	
> 写Batch 047的时候想起有款爱沙尼亚的金酒干了大黄为主要，那味道可是有点难扛hh
> 非常“社区”的一个蒸馏厂，会有许多活动：农民集市、音乐演出，和过山殿一般亲切。
